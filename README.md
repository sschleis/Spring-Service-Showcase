Example REST Service

## Endpoints

**GET** http://localhost:8085/customer/findAll

**GET** http://localhost:8085/customer/<ID>

**POST** http://localhost:8085/customer <br/>
Body:
```
{
    "firstname": "Luke",
    "lastname": "Skywalker",
    "gender": "Male"
}
```

**DELETE** http://localhost:8085/customer/deleteAll
